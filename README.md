# TimerApp4Seminar

## 概要
大阪電気通信大学のゼミナールで使うために作ったタイマーアプリです。


## 主な機能
- カウントアップタイマー（MM:SS）
- 段階警告（ベル音）
- ラップタイム記録・表示
- 設定のURLパラメータ指定・共有
- モバイル/PC両対応、シンプルUI

---

## 使い方

1. サイトにアクセス
2. ▶️（スタート）、⏸️（一時停止）、🔄（リセット）、⏱LAP（ラップ記録）ボタンで操作
3. 下部で各警告の秒数・段階数を自由に入力できます
4. 設定した内容でカウントアップ＆任意タイミングでベルが鳴ります

---

## URLパラメータでの初期設定方法

### パラメータ仕様

- **bell_count**  
  警告段階（ベルを鳴らす回数）を指定します（1～10）
  - 例: `?bell_count=4`

- **w1, w2, …**  
  各警告の秒数（最大10段階、w1=警告1、w2=警告2…）
  - 例: `?w1=600&w2=900&w3=1200`
  - すべて秒数で指定（例：600=10分）

- **併用例**  
  - 例: `?bell_count=3&w1=300&w2=600&w3=900`
    - ⚠️ 必要数（bell_count）に合わせてw1, w2...を指定してください

### パターン例

- 3段階・ベル3回（10分, 15分, 20分で警告）  
  `https://your-app-url?bell_count=3&w1=600&w2=900&w3=1200`

- 5段階・任意時間で警告  
  `https://your-app-url?bell_count=5&w1=300&w2=600&w3=900&w4=1200&w5=1500`

- 秒数のみ指定（ベル段階数はデフォルト=3）  
  `https://your-app-url?w1=720&w2=900&w3=1500`

> 上記URLをそのまま共有することで、受講者やスタッフ間で同じ設定のタイマーをすぐ利用できます。
